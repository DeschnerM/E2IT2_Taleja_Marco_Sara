# E2IT2_Taleja_Marco_Sara
1.	Zuerst muss das gesamte Projekt über GitHub heruntergeladen werden.
	Dazu oben rechts im Repository auf „Code“ klicken und anschließend „Download ZIP“ auswählen.
	Dadurch wird eine ZIP-Datei heruntergeladen, die alle Dateien enthält, die für das Projekt notwendig sind.
2.	Danach muss die heruntergeladene ZIP-Datei entpackt werden.
	Beim Entpacken entsteht ein Ordner, in dem sich alle HTML-Dateien, die CSS-Datei, die Bilder und mögliche Zusatzdateien befinden.
	Dieser Ordner ist der komplette Projektordner.
3.	Diesen Projektordner soll man in einer geeigneten Entwicklungsumgebung öffnen.
	Empfohlen wird Visual Studio Code, weil man dort den gesamten Ordner inklusive aller Dateien sehen kann.
	In Visual Studio Code einfach auf „Ordner öffnen“ klicken und dann den entpackten Projektordner auswählen.
	Dadurch kann der Lehrer den HTML-Code, das CSS und die Struktur der Seite vollständig einsehen und bewerten.
4.	Um die Webseite selbst anzuschauen, soll die Datei index.html geöffnet werden.
	Dazu reicht ein Doppelklick oder ein Rechtsklick „Im Browser öffnen“.
	Wichtig ist, dass die Seite im Google Chrome Browser geöffnet wird, weil die Darstellung dort korrekt getestet wurde.
	Andere Browser können Darstellungselemente eventuell anders anzeigen.
5.	Die Webseite muss auf einem normalen Computerbildschirm geöffnet werden.
	Die Bildschirmgröße sollte mindestens 24 Zoll betragen, und das Seitenverhältnis sollte 16:9 sein.
	Die Seite ist nicht für mobile Geräte programmiert.
	Daher sollen Smartphones, Tablets oder verkleinerte Fenster nicht verwendet werden.
	Der Lehrer soll die Seite in der normalen Desktop-Ansicht öffnen, so wie sie von uns vorgesehen ist.
6.	Für die Bewertung sollen zwei Dinge gemacht werden:
	Erstens der Projektordner in der IDE ansehen, damit der Lehrer den Code und die Struktur kontrollieren kann.
	Zweitens die HTML-Dateien (vor allem index.html, RAM.html, Grafikkarte_Taleja.html und Prozessor-CPU.html) im Chrome-Browser öffnen, damit er das fertige Design, die Navigation, die Bilder, Videos und alle anderen Elemente
